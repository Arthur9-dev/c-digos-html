história

entre 1991 e 1999 o HTML se desenvolveu a partir da versão 1 até a 4. Em 2008 a primeira versão do HTML5 foi lançada

as primeiras versões do HTML foram definidas com regras sintáticas flexíveis o que ajudou aqueles sem familiaridade com a web. Com o tempo aumentou a tendência em tornar a sintaxe cada vez mais rígida. Apesar disso, por questões históricas, os navegadores ainda hoje conseguem interpretar páginas web desenvolvidas com as primeiras versões de HTML
