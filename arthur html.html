O que é HTML?

O HTML (HyperText Markup Language) é a linguagem padrão para criar e projetar páginas da web. Ele define a estrutura básica de uma página web usando uma série de elementos, como cabeçalhos, parágrafos, listas, links, imagens, formulários, entre outros.

Principais Elementos HTML

  • <!DOCTYPE html>: Define a versão do HTML
  • <html>: Elemento raiz
  • <head>: Cabeçalho da página
  • <title>: Título da página
  • <body>: Corpo da página
  • <h1> a <h6>: Cabeçalhos
  • <p>: Parágrafos
  • <a>: Links
  • <img>: Imagens
  • <ul>: Lista não ordenada
  • <ol>: Lista ordenada
  • <li>: Item da lista
  • <form>: Formulários
  • <input>: Entradas de formulário
