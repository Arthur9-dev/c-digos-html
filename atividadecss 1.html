Desenvolvimento web

algumas tecnologias associadas ao desenvolvimento web
HTMLHTML não é uma linguagem de programação; é uma linguagem
de marcação, usada para definir a estrutura do seu conteúdo.
HTML consiste de uma série de elementos, que você usa para
delimitar ou agrupar diferentes partes do conteúdo para
que ele apareça ou atue de determinada maneira.
CSSCSS é a linguagem que usamos para estilizar um documento
HTML. CSS descreve como os elementos HTML devem ser exibidos.
Este tutorial irá ensinar-lhe CSS do básico ao avançado
JAVA SCRIPTJavaScript é uma linguagem de programação que permite
a você implementar itens complexos em páginas web — toda vez
que uma página da web faz mais do que simplesmente mostrar a
você informação estática — mostrando conteúdo que se atualiza
em um intervalo de tempo
PHPPHP (um acrônimo recursivo para "PHP: Hypertext Preprocessor",
originalmente Personal Home Page) é uma linguagem interpretada livre,
usada originalmente apenas para o desenvolvimento de aplicações presentes
e atuantes no lado do servidor
